Falls Sie die Feld-Typen noch nicht genau kennen, treffen Sie eine Auswahl aus der folgenden Liste und entscheiden Sie, welcher Feldtyp am Besten Ihren Anforderungen entspricht.
Text-Eingabefeld

Einzeilige Eingabefelder dienen zur Aufnahme von einem oder wenigen Worten oder einer Zahl.


Mögliche Angaben: anzuzeigende Feldlänge, alpha oder rein numerisch, minimale und maximale Anzahl der Eingabezeichen, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Text-Area

Mehrzeilige Eingabefelder dienen zur Aufnahme von Kommentaren, Nachrichten usw. Um darin einen Zeilenumbruch zu erzeugen, muss am Ende der Zeile <br /> eingegeben werden. Die Eingabetaste hilft nicht.


Mögliche Angaben: minimale Anzahl der Eingabezeichen, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Textarea mit Wysiwyg-Editor

Mit Hilfe dieses Editors erstellen Sie Texte ohne HTML-Kenntnisse. Ihre Eingaben werden später genauso angezeigt, wie Sie sie formatieren. Haben Sie HTML-Kenntnisse, können Sie die HTML über den Button "HTML" editieren.


Mögliche Angaben: minimale Anzahl der Eingabezeichen, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

E-Mail-Eingabefeld

Einzeiliges Eingabefeld zur Aufnahme einer E-Mail-Adresse. Das korrekte Format der Adresse wird geprüft..


Mögliche Angaben: anzuzeigende Feldlänge, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Passwort-Eingabefeld

Einzeiliges Eingabefeld zur Aufnahme eines Kennworts. Die eingegebenen Daten werden verdeckt statt im Klartext angezeigt.


Mögliche Angaben: anzuzeigende Feldlänge, minimale und maximale Anzahl der Eingabezeichen, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Link-Eingabefeld

Einzeiliges Eingabefeld zur Aufnahme einer Internetadresse.


Mögliche Angaben: anzuzeigende Feldlänge, Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Selectbox

Sie können dem Anwender eine Liste mit festen Einträgen anbieten, aus der er genau einen Eintrag auswählen kann.


Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Multi-Selectbox

Sie können dem Anwender eine Liste mit festen Einträgen anbieten, aus der er einen bis mehrere Einträge auswählen kann.


  • Heino
  • Michael Jackson
  • Tom Waits
  • Nina Hagen
  • Marianne Rosenberg

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Bilder-/Textauswahl (Preselect)

Für jeden Wert in der Auswahlliste des Eingabeformulars kann ein "Ersatzwert" vorgegeben werden, z. B. könnte für "Heino" ein Link, ein Foto oder ein Text vorgegeben werden. Dieser "Ersatzwert" wird bei der späteren Ausgabe auf der Internetseite statt "Heino" angezeigt, also ein Link, ein Foto oder ein Text, je nachdem, was definiert wurde.


Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Radio-Button

Radio-Buttons sind eine Gruppe von beschrifteten Knöpfen, von denen der Anwender einen auswählen kann. Es kann immer nur einer der Radio-Buttons ausgewählt sein.

Beispiel Radio-Buttons


Man kann aber auch nur einen Radio-Button vorgeben, um z. B. die AGB zu akzeptieren oder den Newsletter zu abonnieren.

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Checkbox

Checkboxen sind eine Gruppe von ankreuzbaren Quadraten, bei denen der Anwender keine, eins oder mehrere auswählen kann.

Beispiel einer Checkbox

Wird mehr als eine Checkbox benötigt, müssen mehrere Felder hierzu definiert werden, je Checkbox ein Feld.

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Timestamp

Der Timestamp dient zur Angabe eines Datums. Das angegebene Datum wird anschliessend auf Gültigkeit geprüft.

Beispiel eines Timestamp
Geburtsdatum

Das gewählte Datum wird im Frontend in der Form TT.MM.JJJJ ausgegeben.

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

File-Upload-Eingabefeld

Dieses Feld dient zum Hochladen einer Datei. Die Datei kann später zum Download bereitgestellt werden.


{**}

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Bild-Upload-Eingabefeld

Es kann ein Bild hochgeladen werden. Das hochgeladene Bild kann im Frontend angezeigt werden.


{**}

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Bilder-Galerie

Es können beliebig viele Bilder hochgeladen werden. Die hochgeladenen Bilder können im Frontend als Bildergalerie angezeigt werden.


{**}

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Zeitintervall

Es kann ein Zeitraum (Datum von/bis) angegeben werden. Die Gültigkeit der Daten wird geprüft.

Beispiel eines Zeitintervalls
Datum von
bis

Beispiel Veranstaltungskalender: Wird im Formular der Zeitraum der Veranstaltung mit Hilfe der obigen Auswahlen festgelegt, erscheint die Veranstaltung zusammen mit den vorgegebenen Daten des Zeitraums im Frontend. Die Veranstaltung wird jedoch nur im vorgegebenen Zeitraum im Frontend angezeigt, nicht mehr ausserhalb des Zeitraums.

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.

Preisintervall

Beim Preisintervall können mehrere Preise definiert werden, die in der Suchmaske im Frontend zur Suche nach einer Preisspanne dienen (Preis liegt im Bereich von/bis).

Mögliche Angaben: Hilfetext, Fehlertext, Labletext, Feldname für den Export, Fieldset(Gruppen)-Zuordnung.
