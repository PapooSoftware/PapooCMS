Messe-Reminder

Hier können Sie die Erinnerungen für Messeeinträge verwalten.

Klicken Sie links auf Vorlage um die Vorlage, nach der die Einträge für neue Messen erstellt werden, zu bearbeiten.

Unter Einträge finden Sie die Liste der vorhandenen Messen und deren Einstellungen.
Diese werden automatisch erstellt, sobald Sie die Vorlage fertig ausgefüllt haben.

Damit die Erinnerungen versandt werden können, müssen Sie folgenden Link als täglichen Cronjob einrichten:

{$urlbase}template=messe_reminder/templates/cronjob.html&username=Benutzername&password=Passwort
