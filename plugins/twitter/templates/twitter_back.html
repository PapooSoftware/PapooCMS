Twitter Plugin


Mit dem Twitter Plugin kann man sich eine beliebige Anzahl von Tweets oder Followern ueber einen Benutzernamen oder Hashtag anzeigen lassen.

1. Tweets ueber Benutzername anzeigen

Mit #twitter+usertweets lassen sich die letzten Tweets vom jeweiligen Benutzernamen anzeigen.
Die Syntax lautet: #twitter+usertweets+name+count+mediayn#
Ein Beispiel waere #twitter+usertweets+twitterapi+5+0# um sich die letzten 5 Tweets des Benutzers mit dem Namen twitterapi ohne Bilder/Videos ausgeben zulassen.
Wenn Sie den letzten Parameter auf "1" setzen, so werden Medien im tweet mit angezeigt.

2. Tweets ueber Hashtag anzeigen

Mit #twitter+hashtweets lassen sich die letzten Tweets mit jeweiligem Hashtag anzeigen.
Die Syntax lautet: #twitter+hashtweets+name+count+mediayn#
Ein Beispiel waere #twitter+hashtweets+cms+5+0# um sich die letzten 5 Tweets mit dem Hashtag cms ohne Bilder/Videos anzeigen zulassen.
Wenn Sie den letzten Parameter auf "1" setzen, so werden Medien im tweet mit angezeigt.

3. Followers ueber Benutzername anzeigen

Mit #twitter+followers lassen sich die letzten Tweets mit jeweiligem Hashtag anzeigen.
Die Syntax lautet: #twitter+followers+name+count#
Ein Beispiel waere #twitter+followers+twitterapi+30# um sich 30 Followers vom Benutzer mit dem Namen twitterapi anzeigen zulassen.

4. Einen bestimmten Tweet einbinden

Wenn Sie einen bestimmten Tweet einbinden möchten, kopieren Sie den Link zum Tweet, wie im folgenden Bild dargestellt und fügen ihn an der gewünschten Stelle wie angegeben ein.
